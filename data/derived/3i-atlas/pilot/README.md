# DERIVED Packages

This directory contains DERIVED packages produced by Gate 4A pilot workflows.

Each package is stored in a subdirectory with UTC timestamp:
`as_of_utc=YYYY-MM-DDTHH:MM:SSZ/`

Packages are produced by the Gate 4A pilot workflow (manual-only):
`.github/workflows/gate-4a-pilot.yml`

See `docs/gate-4a/README.md` for details.
